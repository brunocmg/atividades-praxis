Mini guia Git/GitHub

Lista de alguns comandos úteis

  1. git init - Inicializa um repositório Git local
  2. git clone - Cria uma cópia local de um repositório remoto
  3. git status - Checa o status
  4. git add (nome do arquivo.html) - Adiocionar arquivo especifico
  5. git add . - Adiciona os arquivos modificados
  6. git commit -m "(mensagem de commit)" - Comita as alterações
  7. git branch - Lista as branches
  8. git push - Envia as alterações para o repositório remoto
  9. git pull - Atualiza o repositório local para o último commit
  10. git log - Ver modificações
